# Week 9: Files

## Overview

Welcome to Week 9! File handling lets your programs read and write data, making them truly useful for real-world applications.

## Learning Objectives

- Open, read, write, and close files
- Use context managers (`with` statement)
- Handle different file modes
- Work with CSV files
- Process text files line by line
- Handle file errors gracefully

## Topics Covered

1. **File Basics**
   - Opening and closing files
   - File modes (r, w, a, r+, etc.)
   - The `with` statement

2. **Reading Files**
   - `read()` - Read entire file
   - `readline()` - Read one line
   - `readlines()` - Read all lines as list
   - Iterating over files

3. **Writing Files**
   - Writing text
   - Appending to files
   - Writing lists

4. **CSV Files**
   - Reading CSV with csv module
   - Writing CSV files
   - Working with headers

5. **Practical Applications**
   - Log files
   - Data processing
   - Configuration files
   - Reports
