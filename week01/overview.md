# Week 1: Course Introduction & Computer Programming Basics

## Overview

Welcome to your first week! This week we'll introduce you to the course, get you set up with the necessary tools, and start exploring the basics of computer programming with Python.

## Learning Objectives

By the end of this week, you will be able to:

- Navigate the course Canvas site and locate resources
- Install and launch Anaconda/Jupyter Notebook
- Understand what computer programming is and why it's valuable
- Create and run your first Python program
- Understand variables and how to name them properly
- Write comments to document your code
- Follow Python style guidelines (PEP 8)

## Topics Covered

1. **Course Introduction**
   - Syllabus review
   - Course structure and expectations
   - Grading policies
   - Resources and textbooks

2. **Computer Programming Basics**
   - What is programming?
   - Why learn Python?
   - The programming mindset

3. **Getting Started**
   - Installing Anaconda
   - Launching Jupyter Notebook
   - Understanding the interface
   - Creating your first notebook

4. **Variables and Naming Conventions**
   - What are variables?
   - Assigning values
   - Variable naming rules and best practices
   - PEP 8 style guidelines
   - Comments and documentation
