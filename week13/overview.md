## Key Concepts

```{important}
**Robotics = Sensing + Deciding + Acting**

This is the core loop:
1. **Sense** - Get information about the environment
2. **Decide** - Process data and choose action
3. **Act** - Execute the action
4. **Repeat** - Loop continuously
```
